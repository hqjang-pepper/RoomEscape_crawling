##README
###프로젝트의 목적
> 한국 주요 방탈출 카페 8개의 사이트로부터 예약 현황 및 
###파일 설명
* day.txt
> 크롤링할 날짜를 설정해 줍니다. 1(오늘)~7(오늘포함7일)까지 한 자리 숫자만 입력해주세요.
* main.py
> 이 파일을 실행시키면 순차적으로 7개의 방탈출 사이트를 크롤링해서 연결된 mysql 서버로 저장이 가능합니다.

### 주의 사항
* 비트포비아(xphobia.py)는 사이트가 크롤링하는데 굉장히 느리게 설계가 되어있습니다. 안하시길 추천드립니다.
* 방탈출사이트.py에는 각 사이트 별로 크롤링 할 수 있게 하는 코드인데, mysql이 제 로컬 서버로 설정되어 있습니다. 혹시 쓰실 분은 각자 환경에 맞게 pymysql connection 부분을 수정 바랍니다.
